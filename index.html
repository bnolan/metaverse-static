Goals

The goal of this project is to create a multiuser 3d world for sharing art and experiences with other people.

Try it now

Create a user name to join the world and explore other peoples art. If you want to upload your own art into the world, read our art importers guide.

Components

Asset server - server

The asset server accepts many kinds of 3d models for upload, and processes them so they can be loaded by the client javascript. Currently supported models are:

.3ds, .dae, .blender, .obj

The asset server currently outputs three.js comopatible .js models. The asset server uses CORS so that any client can drag and drop models into the world.

Scene Server - github

The scene server is a node.js app that simulates the world, runs javascript code to modify the world state, and shares the world state with everyone who is connected. The scene server is designed to be cross platform and run on Windows and Mac.
